# Base curricular y documental del repositorio

## Finalidad

Este repositorio base se ha construido para ofrecer materiales iniciales editables desde Infantil hasta Secundaria.
No pretende sustituir una programacion completa de centro, sino servir como punto de arranque para que el profesorado adapte contenidos, proyectos, fichas y criterios a su realidad de aula.

## Referencias editoriales usadas solo como inspiracion estructural

Tambien se han observado patrones publicos de presentacion de proyectos editoriales de uso escolar, como los de SM o Anaya, unicamente para extraer rasgos de organizacion:

- apertura con una situacion significativa;
- secuencia por unidades o situaciones de aprendizaje;
- combinacion de saberes, tareas y producto final;
- evaluacion formativa y evidencias;
- atencion a la diversidad y propuesta transferible al aula.

No se ha copiado redaccion, actividades cerradas ni maquetaciones completas de ningun manual comercial.

## Criterios de redaccion de esta base

- Texto original y editable.
- Estructura modular para que el profesorado pueda borrar, ampliar o reorganizar.
- Lenguaje docente claro, sin dependencia de una comunidad autonoma concreta.
- Foco competencial y metodologico.
- Inclusiones sencillas de producto final, evaluacion y adaptaciones.

## Matriz base del catalogo

### Infantil

- Lengua Infantil: oralidad, cuentos, rimas y primeras practicas de lectura y escritura.
- Matematicas Infantil: pensamiento logico, patrones, conteo y resolucion de retos manipulativos.
- Conocimiento del Medio Infantil: observacion del entorno, autonomia, cuerpo y primeros proyectos.

### Primaria

- Lengua Primaria
- Matematicas Primaria
- Conocimiento del Medio Primaria
- Ingles Primaria
- Educacion Artistica Primaria

### Secundaria

- Lengua Secundaria
- Matematicas Secundaria
- Conocimiento del Medio Secundaria como libro interdisciplinar de apoyo
- Geografia e Historia Secundaria
- Biologia y Geologia Secundaria
- Fisica y Quimica Secundaria
- Ingles Secundaria
- Tecnologia y Digitalizacion Secundaria

## Uso recomendado

1. Duplicar el libro base en una rama docente.
2. Ajustar saberes, temporalizacion y criterios de evaluacion del centro.
3. Crear fichas y recursos concretos para cada grupo.
4. Registrar cambios y propuestas de mejora desde el propio editor.
