# Repositorio base de Libre Libros

Este repositorio contiene una base de libros editable para Infantil, Primaria y Secundaria.
La idea no es replicar manuales comerciales ni copiar materiales ajenos, sino ofrecer un punto de partida original para que los equipos docentes puedan adaptarlo a su centro, su etapa y su programacion.

## Criterios de construccion

- Alineacion con el marco curricular estatal vigente en Espana.
- Redaccion original, sin copiar textos de editoriales ni de otros libros.
- Uso de recursos educativos abiertos como referencia metodologica y de estructura.
- Organizacion tipo libro docente: proposito, situaciones de aprendizaje, evaluacion, inclusion y producto final.

La guia de referencia esta en [books/BASE_CURRICULAR.md](books/BASE_CURRICULAR.md).

## Estructura de cada libro

Cada libro vive en su propia carpeta y puede incluir:

- `book.md`
- `assets/`
- `worksheets/`

Los contenidos estan pensados para abrirse tal cual en Libre Libros y servir de base de trabajo para crear versiones de aula, ramas docentes, fichas y recursos complementarios.
